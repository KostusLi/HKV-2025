<h1>Мой курсовой проект по КПО</h1>
Вариант 29 - 731 10 2

Основные задания:
1. Типы данных: целочисленный(1-байтовый) и символьный;
2. Функции стандартной библиотеки: сравнение 2-х строк, произвольная;
3. Побитовые операции: И, ИЛИ, НЕ(инверсия);
4. Литералы: целый - в десятичном и восьмеричном представлении, строковые и/или символьные - в произвольном;
5. Условный оператор: switch;

Дополнительные задания:
1. Реализован строковый тип данных;
2. Арифметические операции: + - / * %;
3. Цикл(patrol), if-else(charge-backup);
4. Функции стандартной билиотеки: вычисление длины строки, конкатенация 2-х строк, рандомное число из промежутка, преобразование строки в число;
